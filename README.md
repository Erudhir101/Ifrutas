## Execução do Projeto

Instalar o Projeto com **GIT**
```bash
git clone https://github.com/Erudhir101/Ifrutas.git
```
Vá para dentro do **Projeto**
```bash
cd /IFrutas
```
Instale todas as **dependências**
```bash
npm install
```
execute o projeto com **Expo**
```bash
npx expo start
```
